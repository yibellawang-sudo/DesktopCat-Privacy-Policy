Privacy Policy for Desktop Cat Effective Date: March 4th, 2026 Desktop Cat does not collect, transmit, or share any personal information or user data. Desktop Cat operates entirely locally within the user’s browser. The extension: - Does not collect personally identifiable information - Does not collect browsing history - Does not track user activity - Does not collect location data - Does not transmit data to external servers The extension uses Chrome’s storage API to store: - User customization settings (such as selected cat type and position) - User-created reminder times and intervals This data is stored locally on the user’s device and is never transmitted or shared. Third Parties: Desktop Cat does not use analytics services, advertising services, or third-party tracking tools. Changes: If the extension’s data practices change in the future, this privacy policy will be updated accordingly.
